순서 있는 목록 태그 ol

블록 요소 / 자동으로 숫자가 부여 됨.

  1. start 속성 작성시 작성한 숫자부터 시작.
  2. ol과 li 사이엔 다른 태그가 올 수 없다.
  3. li태그 안에는 다양한 다른 태그들 작성 가능.
  4. ol과 li는 함께 사용. 둘 중 하나만 사용할 수 없다.
  5. 꼭 순서가 필요한 리스트에 작성.
  1. 감자를 1/4 크기로 자른다.
  2. 부드러워질 때까지 15분-20분 동안 소금물에 서서히 끓인다.
  3. 감자의 물을 따라 내고 으깬다.
  4. 우유, 버터, 육두구를 데운다.
  5. 우유 혼합물에 넣고 섞는다.
