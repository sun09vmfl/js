요소의 실제 크기 box size1

width + margin-left + margin-right + padding-left + padding-right + border-left + border-right

height + margin-top + margin-bottom + padding-top + padding-bottom + border-top + border-bottom

width + margin + padding + border

.box1의 실제 가로 크기: w500+m(20+20)+p(10+10)+b(10+10)=580px

.box1의 실제 세로 크기: h300+m(20*2)+p(10*2)+b(10*2)=380px

width + margin + padding + outline(이 특성은 요소의 실제 크기에 영향을 주지 않음)

.box2의 실제 가로 크기: w500+m(20+20)+p(10+10)=560px

.box2의 실제 세로 크기: h300+m(20*2)+p(10*2)=360px
